# Hello World Skill

When invoked, greet the user warmly and share one fun programming fact.

## Triggers

- User types `/hello <name>`
- User says "hello, I'm <name>" or "hi, my name is <name>"

## Response format

1. Greet the user by name (e.g. "Hey Alice!")
2. Share one fun programming fact (rotate through different ones)
3. Offer to help with a coding task

## Example

**User:** /hello Alice

**Claude:** Hey Alice! Did you know that the first computer bug was an actual bug — a moth found trapped in a relay of the Harvard Mark II computer in 1947? Anyway, what can I help you build today?
