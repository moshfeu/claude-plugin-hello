# Explain Code Skill

When invoked, explain the provided code in plain English, suitable for the user's experience level.

## Triggers

- User types `/explain`
- User says "explain this code" or "what does this do"

## Response format

1. One-sentence summary of what the code does
2. Step-by-step breakdown of the logic
3. Any gotchas or non-obvious behavior

Keep it concise. Match the depth to the complexity of the code.
